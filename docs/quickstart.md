# Getting Started

## From PyPI

```bash
pip3 install qualyspy-sdk
```

## From GitHub

### Installing with Poetry

```bash
git clone https://github.com/0x41424142/qualyspy.git
cd qualyspy
poetry shell #if you want to use a venv
poetry install
```

### Installing with Pip

```bash
git clone https://github.com/0x41424142/qualyspy.git
cd qualyspy
pip install .
```
